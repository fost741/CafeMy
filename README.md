# CafeMY: Система обліку для кафе

## Суть проєкту
**CafeMY** — це консольний застосунок на C#, призначений для автоматизації роботи невеликого кафе. Програма дозволяє ефективно формувати рахунки, розраховувати підсумкову вартість товарів, додавати чайові, враховувати податки (GST) та забезпечує можливість збереження/завантаження даних через файлову систему.

## Основний функціонал

*   **Управління рахунком:** Додавання товарів (з перевіркою валідності назви та ціни) та видалення позицій за індексом.
*   **Гнучкі налаштування чайових:** Розрахунок чайових через відсоткову ставку або введення фіксованої суми.
*   **Фінансові розрахунки:** Автоматичне обчислення проміжної суми (`Net Total`), податку (`GST`) та фінального підсумку.
*   **Робота з файлами:** Збереження сформованого чека у форматі `.txt` та відновлення даних рахунку з файлу.
*   **Валідація:** Перевірка введених даних, робота з діапазонами значень та очищення рахунку.

## Структура файлу (*.txt)
Збережений файл містить табличний звіт:
1. **Шапка:** Заголовки «Description» та «Price» + лінія розділення.
2. **Список:** Рядки з назвами товарів (ліворуч) та цінами (праворуч).
3. **Підсумок:** Роздільна лінія, розрахунки (`Net Total`, `Tip Amount`, `GST Amount`, `Total Amount`) та фінальна лінія.

## Технічний стек
*   **Мова:** C# (.NET)
*   **Інтерфейс:** Консольний (CLI)
*   **Формат даних:** Текстові файли (`.txt`)
